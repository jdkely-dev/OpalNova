# OPALNOVA V1.83.0 Supplier Diamond Replacement Readiness Testing Checklist

Use this checklist after launching the V1.83.0 build.

- [ ] Confirm the main header shows `Version 1.83.0 - supplier diamond replacement readiness`.
- [ ] Open About and confirm it shows `Version 1.83.0 - Supplier Diamond Replacement Readiness`.
- [ ] Open Release Notes and confirm V1.83.0 appears above V1.82.0.
- [ ] Open Supplier Diamond Holds & Orders.
- [ ] Select a saved supplier diamond row.
- [ ] Confirm the selected diamond details still show certificate, quote, customer, status and lifecycle.
- [ ] Click `Copy Replacement Search`.
- [ ] Paste into a text editor and confirm the copied text includes type, shape, carat range, colour/clarity, lab, original certificate and quote/customer context.
- [ ] If close saved alternatives exist, confirm up to five are listed with certificate, price, currency and status.
- [ ] If no close saved alternatives exist, confirm the copied text tells you to run a supplier search with the copied criteria.
- [ ] Confirm Request Hold, Confirm Hold, Request / Mark Ordered, Mark Received, Convert To Owned Stone and Create Reminder Task still open/respond normally.
- [ ] Confirm no supplier credentials are displayed or hardcoded in the replacement text.
- [ ] Close OPALNOVA cleanly.
