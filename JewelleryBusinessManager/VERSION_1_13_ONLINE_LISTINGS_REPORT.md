# OPALNOVA V1.13 — Online Listing & Content Tracker

## Summary
V1.13 adds online listing workflow tools for finished jewellery stock.

## Added
- Online Listings section
- OnlineListing model and upgrade-safe SQLite table creation
- Listing status and photo status enums
- Create Listing workflow button
- Generate Content workflow button
- Listing Checklist printout
- Listing Report printout
- Dashboard cards for Needs Listing, Ready To List, and Listed Online
- Export bundle and health check coverage for online listings

## Workflow
1. Select a Jewellery Stock item.
2. Click Create Listing.
3. Click Generate Content to fill SEO title, descriptions, caption and hashtags.
4. Open Online Listings to edit platform, URL and checklist fields.
5. Click Listing Checklist to create a printable listing preparation sheet.
6. Click Listing Report to audit items needing photos/content/listing.

## Validation
Two static validation passes were run with zero blocking errors and zero warnings.
