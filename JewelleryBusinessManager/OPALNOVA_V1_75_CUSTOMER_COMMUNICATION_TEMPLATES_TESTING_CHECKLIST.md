# OPALNOVA V1.75.0 Customer Communication Templates Testing Checklist

Use this checklist after launching the V1.75.0 build.

- [ ] Confirm the main header shows `Version 1.75.0 - customer communication templates`.
- [ ] Open About and confirm it shows `Version 1.75.0 - Customer Communication Templates`.
- [ ] Open Customer Relationship Studio.
- [ ] Select a customer and run Communication Templates.
- [ ] Confirm the generated report includes customer contact, value guidance and suggested next step.
- [ ] Confirm the report includes relevant message starters for the customer's current context.
- [ ] Open Customer Summary Card for the same customer and confirm lifetime value and relationship guidance appear.
- [ ] Open Customer Timeline and confirm lifetime value and suggested next step appear.
- [ ] Run Relationship Report and confirm lifetime value, value guidance and suggested next step columns appear.
- [ ] Create Customer Follow-Up and confirm follow-up notes include value guidance and a message starter.
- [ ] Confirm Customer History, Customer Summary Card, Customer Timeline and Relationship Report still open normally.
- [ ] Confirm no database migration prompt or schema reset occurs.
