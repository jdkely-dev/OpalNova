# OPALNOVA V1.62.0 - Built-In Help Manual Refresh

## Build Scope

V1.62.0 expands the built-in User Guide from a short routine page into a practical local manual for day-to-day OPALNOVA use.

## Changes

- Expanded `DataSafetyService.CreateUserGuide()`.
- Added sections for:
  - setup and daily rhythm.
  - quotes and proposals.
  - production workflow.
  - payments, invoices and handover.
  - inventory and supplier diamonds.
  - reports and bookkeeping review.
  - backups, restore and data safety.
  - release testing routine.
- Added cautions for accepted quotes, job completion, supplier stock, restore, import and bulk updates.
- Kept the guide generated locally as HTML from the existing User Guide action.
- Updated visible version text, release notes, About text, roadmap and handoff notes.

## Data Safety

- No database schema changes.
- No business records are read or modified to generate the guide.
- Existing User Guide entry points remain unchanged.

## Notes

- This is a practical operating manual, not a full searchable help system.
- Searchable/paginated help can be added later after more workflow text stabilizes.
