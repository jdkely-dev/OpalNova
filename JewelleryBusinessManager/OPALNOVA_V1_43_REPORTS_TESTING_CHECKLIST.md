# OPALNOVA V1.43 Reports Testing Checklist

Open OPALNOVA, then go to **Reports Studio** and test:

- [ ] BI Command Report opens in the in-app report viewer.
- [ ] Weekly Sales opens and totals make sense.
- [ ] Monthly Sales opens and totals make sense.
- [ ] Outstanding Balances shows jobs with balances owing.
- [ ] Quote Conversion shows quote status and accepted/linked jobs.
- [ ] Inventory Value opens and includes jewellery, stones and materials.
- [ ] Reserved Inventory opens and shows quote reservations.
- [ ] Customer Follow-Ups opens and shows open tasks/follow-ups.
- [ ] Opal / Stone Stock opens and shows stone details.
- [ ] Export BI CSV creates a folder containing CSV files.
- [ ] Open HTML works from the report viewer.
- [ ] Open Folder works from the report viewer.
- [ ] Print from the browser/report preview works.
- [ ] Publish script still creates OPALNOVA.exe.
- [ ] Published OPALNOVA.exe opens normally.

Reports are read-only snapshots. They should not change records.
