# OPALNOVA V1.45 External Diamond Quote Linking Testing Checklist

1. Open Diamond Supplier Studio.
2. Search Nivoda supplier diamonds using user-entered credentials.
3. Save at least one diamond result.
4. Open Custom Quote & Proposal Workflow.
5. Create or open a quote.
6. Select a quote option.
7. In External supplier diamonds, select the saved external diamond.
8. Confirm the supplier cost fills in.
9. Click Link External Diamond.
10. Confirm the stone cost and live total update.
11. Save, close and reopen the quote.
12. Confirm the external diamond link is still shown.
13. Generate Preview Proposal and confirm the external diamond details appear.
14. Change the linked external diamond status to Hold Requested or Hold Confirmed.
15. Save and reopen the quote.
16. Accept the quote option.
17. Create / update the production job.
18. Open the job and confirm the external diamond details are included in design/job notes.
19. Publish standalone and confirm OPALNOVA.exe opens normally.

Notes:
- External supplier diamonds are not owned stock.
- They should not deduct from OPALNOVA stone inventory.
- Confirming availability, hold, order and payment should still be done through the supplier until production integration is expanded.
