# OPALNOVA V1.65.0 Quote Context Testing Checklist

Use this checklist after launching the V1.65.0 build.

- [ ] Confirm the main header shows `Version 1.65.0 - quote context fields`.
- [ ] Open About and confirm it shows `Version 1.65.0 - Quote Context Fields`.
- [ ] Open Custom Quote Builder.
- [ ] Create or open a quote and fill Occasion, Required by, Ring size, Budget / target, Preferred metal and Preferred stone.
- [ ] Add text to Internal notes and save the quote.
- [ ] Reopen the quote and confirm all new fields reload correctly.
- [ ] Preview the proposal and confirm customer-facing project details appear when recorded.
- [ ] Confirm Internal notes do not appear in the proposal output.
- [ ] Set Required by to today or within seven days and confirm quote next-action guidance warns about timing.
- [ ] Open Alert Centre or Project Workbench and confirm required-by quote risks appear when relevant.
- [ ] Open Proposal Pipeline and confirm quote context appears in selected-row details and can be found by search.
- [ ] Confirm existing quotes with blank context fields still open and save normally.
- [ ] Close the app normally.
