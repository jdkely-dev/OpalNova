# OPALNOVA V1.68.0 Balance Reminder Testing Checklist

Use this checklist after launching the V1.68.0 build.

- [ ] Confirm the main header shows `Version 1.68.0 - balance reminder workflow`.
- [ ] Open About and confirm it shows `Version 1.68.0 - Balance Reminder Workflow`.
- [ ] Open Payment & Collection.
- [ ] Select a job with a balance owing.
- [ ] Click Copy Balance Reminder and paste the clipboard text into a blank note or message window.
- [ ] Confirm the copied text includes job code/title, total, paid amount, remaining balance and due date if available.
- [ ] Click Create Balance Follow-Up and confirm a task is created.
- [ ] Click Create Balance Follow-Up again for the same job and confirm OPALNOVA prevents a duplicate open task.
- [ ] Select a job with no balance owing and confirm reminder/follow-up actions show a clear information message.
- [ ] Confirm Record Payment, Generate Invoice / Receipt and handover buttons still work as before.
- [ ] Close the app normally.
