# Version 1.25 — Hardware/POS UI Refinement

## Focus
This version refines the presentation and usability of the hardware and market operations workflows without changing the confirmed-working business logic.

## Updated areas
- DYMO / Mini Label Studio
- Camera & Scale Capture Studio
- Market Operations Studio
- Customer Display window

## Improvements
- More modern gradient headers
- Clear step-by-step workflow cards
- Stronger visual hierarchy
- Larger primary actions
- Better status/help panels
- More polished live previews
- Cleaner operator/customer-display separation
- Sharper text rendering settings
- Improved market POS layout with dedicated setup, checkout, stock, and preview areas

## Behaviour preserved
- DYMO label preview, print dialog and HTML fallback
- Scale reading / manual weight entry
- Weight apply to materials and stones
- Photo import and camera-roll import
- Market operator window
- Customer display lifecycle fix
- Market sale recording
- Backup/export/restore features
