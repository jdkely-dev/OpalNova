# OPALNOVA V1.99.0 Pre-Milestone Hardening Testing Checklist

Use this checklist against the published V1.99 build before treating it as the V2.0 release-candidate baseline.

- Launch OPALNOVA and confirm the header shows `Version 1.99.0 - pre-milestone hardening`.
- Open About and confirm it shows `Version 1.99.0 - Pre-Milestone Hardening`.
- Open Customer Relationship Studio and confirm `Customer Timeline` mini help opens the Customer Timeline-specific guide.
- Open Customer Relationship Studio and confirm `Communication Templates` mini help still opens the Communication Templates guide.
- Open Settings & Backup and run `Release Notes`; confirm V1.99.0 appears above V1.98.0.
- Open Settings & Backup and run `User Guide`; confirm the manual version shows V1.99.0.
- Open Settings & Backup and run `Support Snapshot`; confirm installed version context reports the current 1.99 build.
- Open Safety & Data Studio and run `Release Readiness`; confirm the report opens in preview.
- Reopen the V1.94-V1.98 changed workflows: Payment & Collection, Production Board, Inventory Intelligence, Operations Performance and Support Snapshot.
- Confirm no new action creates duplicate tasks, payments, stock movements, sales or supplier diamond state changes.
- Confirm Debug build, Release publish and published launch smoke have passed before treating this as release-ready.
