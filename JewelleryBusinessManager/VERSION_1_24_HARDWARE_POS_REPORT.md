# Version 1.24 — Hardware Devices, Mini Labels & Market POS

## Added

- Hardware & POS Studio workspace.
- DYMO / mini label printing window using Windows print dialog.
- Printable mini label HTML fallback.
- Camera & precision scale capture window.
- Serial/COM scale support via System.IO.Ports.
- Keyboard-wedge/manual scale reading fallback.
- Apply scale weight directly to selected Materials or Stones.
- USB camera workflow using Windows Camera plus Camera Roll import.
- Photo import linked to selected record.
- Separate Market Operations window.
- Optional customer-facing display window for multi-monitor market POS.
- Market sale workflow that updates Sales, Jewellery Stock and Market Stock.
- Device setup notes report.

## Notes

DYMO direct printing uses Windows printing. The DYMO printer driver or DYMO Connect software should be installed on the PC. Serial scales can be read through COM ports. HID-only scales may need manufacturer software or wedge software to expose readings as keyboard input or a virtual COM port.
