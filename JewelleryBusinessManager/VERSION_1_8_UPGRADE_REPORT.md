# Version 1.8 Upgrade Report

## Added

- Restore Backup workflow with automatic safety backup before restore.
- Database Health Check report.
- Export Full Data Bundle ZIP with database, settings, photos and CSV exports.
- Import CSV into active section with matching field headers.
- Error logging service and Error Log button.
- About/version screen.
- Built-in HTML user guide.
- App-level unhandled exception logging.

## Compatibility

No model or database schema changes were made. The upgrade is intended to work with existing V1.7 data.
