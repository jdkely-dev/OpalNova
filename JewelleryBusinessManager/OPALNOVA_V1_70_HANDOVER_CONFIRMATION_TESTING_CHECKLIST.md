# OPALNOVA V1.70.0 Handover Confirmation Testing Checklist

Use this checklist after launching the V1.70.0 build.

- [ ] Confirm the main header shows `Version 1.70.0 - handover confirmation document`.
- [ ] Open About and confirm it shows `Version 1.70.0 - Handover Confirmation Document`.
- [ ] Open Payment & Collection.
- [ ] Select a job that is ready for collection or shipping.
- [ ] Add a short note in Collection / Shipping Notes.
- [ ] Click Generate Handover Confirmation.
- [ ] Confirm the generated document opens in the default browser.
- [ ] Confirm the document shows customer details, job details, total, paid, balance and linked payment history.
- [ ] Confirm the checklist wording matches collection or shipping where possible.
- [ ] Confirm the handover notes appear in the document.
- [ ] Confirm generating the document does not record a payment, create a sale or complete the job.
- [ ] Confirm Record Payment, Generate Invoice / Receipt, Create Sale From Job and completion buttons still work as before.
