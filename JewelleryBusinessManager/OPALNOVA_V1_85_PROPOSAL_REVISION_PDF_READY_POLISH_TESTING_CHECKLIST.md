# OPALNOVA V1.85.0 Proposal Revision PDF-Ready Polish Testing Checklist

Use this checklist after launching the V1.85.0 build.

- [ ] Confirm the main header shows `Version 1.85.0 - proposal revision PDF-ready polish`.
- [ ] Open About and confirm it shows `Version 1.85.0 - Proposal Revision PDF-Ready Polish`.
- [ ] Open Release Notes and confirm V1.85.0 appears above V1.84.0.
- [ ] Open Custom Quote Builder.
- [ ] Open or create a quote with at least one quote option.
- [ ] Click Preview Proposal and confirm the generated HTML filename includes `Proposal_v001` or the next revision number.
- [ ] Confirm the proposal header shows generated timestamp and revision label.
- [ ] Confirm the proposal has a `Print / Save as PDF` button.
- [ ] Use browser print preview and confirm Save as PDF / Microsoft Print to PDF is available.
- [ ] Generate the proposal again and confirm the next file uses the next revision number.
- [ ] Click Send / Record Proposal and confirm `Copy PDF Steps` is available.
- [ ] Click `Copy PDF Steps`, paste into a text editor and confirm the copied text includes the proposal path and browser print-to-PDF steps.
- [ ] Confirm Copy Email Draft and Open Email Draft still include the generated proposal file path.
- [ ] Record sent and confirm proposal sent/follow-up behaviour still works.
- [ ] Close OPALNOVA cleanly.
