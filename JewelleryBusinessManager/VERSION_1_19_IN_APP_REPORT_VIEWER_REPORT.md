# Version 1.19 — In-App Report Viewer

This version changes report/document output so generated HTML files open inside the main program in a dedicated report viewer panel.

## Added
- In-app report viewer panel
- Generated report title/path display
- Back button to return to the active section
- Open HTML / Print button to open the generated HTML file in the default browser for easy printing
- Open Folder button for locating saved report files

## Preserved
- Existing HTML generation remains intact
- Printable browser-based HTML files are still created
- Existing reports, documents, backup/export/restore, dashboard, tasks, purchasing, market, online listing and opal workflows remain available
