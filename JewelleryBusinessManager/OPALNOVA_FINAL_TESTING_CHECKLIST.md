# OPALNOVA Final Testing Checklist

Use this checklist before calling the app ready for daily business use.

## Open and navigation

- [ ] App opens from Visual Studio.
- [ ] App opens from the published `OPALNOVA.exe`.
- [ ] Dashboard loads.
- [ ] Workspace navigation works.
- [ ] Tool Studio navigation works.
- [ ] Search, Search All, Clear, and Refresh work.

## Records

- [ ] Add customer.
- [ ] Edit customer.
- [ ] Delete test customer.
- [ ] Add inventory item.
- [ ] Edit inventory item.
- [ ] Add job or sale.
- [ ] Add task.
- [ ] Close and reopen app.
- [ ] Confirm saved records remain.

## Help badges

- [ ] Hover over `?` badge shows mini guide.
- [ ] Moving away closes unpinned guide.
- [ ] Clicking badge pins guide.
- [ ] Clicking same badge closes guide.
- [ ] Main button still performs its normal action.

## Reports, documents, and data safety

- [ ] Reports open.
- [ ] Documents/tools open.
- [ ] Backup creates successfully.
- [ ] Backup file can be found in the chosen location.
- [ ] Restore process is understood before relying on it.

## Publish

- [ ] `publish_release_win_x64.bat` runs.
- [ ] Published app opens.
- [ ] Published app saves and reloads data.
- [ ] Final published folder is copied to a safe location.
