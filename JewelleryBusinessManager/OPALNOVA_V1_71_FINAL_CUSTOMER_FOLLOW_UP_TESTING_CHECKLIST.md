# OPALNOVA V1.71.0 Final Customer Follow-Up Testing Checklist

Use this checklist after launching the V1.71.0 build.

- [ ] Confirm the main header shows `Version 1.71.0 - final customer follow-up`.
- [ ] Open About and confirm it shows `Version 1.71.0 - Final Customer Follow-Up`.
- [ ] Open Payment & Collection.
- [ ] Select a job linked to a customer.
- [ ] Click Create Thank-You Follow-Up.
- [ ] Confirm OPALNOVA creates a follow-up task and shows the created task code in the status bar.
- [ ] Click Create Thank-You Follow-Up again for the same job and confirm OPALNOVA prevents a duplicate open task.
- [ ] Open the work queue/task list and confirm the task is visible.
- [ ] Confirm the task is linked to the selected job and customer where available.
- [ ] Confirm the follow-up notes contain a customer-ready thank-you / after-care message.
- [ ] Confirm Generate Handover Confirmation, Create Pickup Reminder, Record Payment, Create Sale From Job and completion actions still work as before.
