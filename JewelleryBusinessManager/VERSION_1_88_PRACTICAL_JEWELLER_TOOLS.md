# OPALNOVA V1.88.0 Practical Jeweller Tools

V1.88.0 starts the practical jeweller tools pass with local calculators and reference data that do not depend on hardware devices.

## Implemented

- Bumped visible/project version metadata to `1.88.0`.
- Added `JewellerToolsWindow`.
- Added a ring-size reference grid covering common AU/UK, US, EU, inside-diameter and inside-circumference values.
- Added a metal-weight estimator using length, width, thickness and metal density.
- Added a faceted-stone carat estimator using shape, length, width, depth and shape-specific factors.
- Added copyable result text for quote notes or manual checking.
- Added Jeweller Tools actions in:
  - Pricing Studio.
  - Hardware & POS Studio.
  - Search All workflow actions.
- Preserved database schema and existing pricing helper, camera/scale capture, DYMO label and market POS behavior.

## Deferred

- Hardware-specific improvements remain dependent on confirmed Windows printer, camera and scale setup.
- Setting-cost, casting-cost and polishing checklist depth remain future additions after the basic tool surface is validated.
