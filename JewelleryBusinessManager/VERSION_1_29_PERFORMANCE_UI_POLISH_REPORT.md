# Version 1.29 — Performance & UI Polish

## Built on
V1.28 condensed navigation confirmed working build.

## Improvements
- Added debounced active-section search to reduce refresh work while typing.
- Added wait cursor during refresh/load operations for clearer feedback.
- Added measured load-time feedback in the status bar for record sections.
- Enabled DataGrid row and column virtualization with recycling for larger data sets.
- Enabled deferred scrolling for large tables.
- Switched generic record table loading to AsNoTracking for faster read-only table views.
- Added small page-level status chips for optimised tables, debounced search and keyboard workflow.
- Refined global theme resources and table rendering settings while preserving the confirmed V1.28 UI.

## Preserved
- Dashboard clickable tiles.
- Condensed navigation.
- Tool workspaces.
- Detail preview panels.
- Backup, restore and export bundle fixes.
- Hardware/POS, barcode labels, tasks, purchase orders, market pro, online listings and opal workflow.
